Free Rider HD Track Mover

by Polygon

This super duper ultra high quality webpage moves Free Rider HD tracks. Paste the exported track code into the big text box, put the coordinates for how much you want to move the track into the X and Y boxes, and press move.

Each number you put in the X and Y boxes is one thenth of a grid unit in the game. Positive X values move the map to the right, and positive Y values move the map up.
